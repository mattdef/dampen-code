# Changelog

All notable changes to the "dampen-vscode" extension will be documented in this file.

## [0.1.0] - 2026-01-30

### Added
- Initial release of Dampen VS Code extension
- Syntax highlighting for `.dampen` files
- LSP client integration with `dampen-lsp`
- Automatic LSP binary download and caching
- Code snippets for common Dampen widgets (button, column, row, text, container)
- Property snippets (padding, margin, width, height, color, backgroundColor)
- Commands: "Dampen: Check Syntax" and "Dampen: Restart Language Server"
- Configuration options: `dampen.lspPath`, `dampen.autoDownload`, `dampen.lspVersion`
- Support for Windows, macOS, and Linux platforms
